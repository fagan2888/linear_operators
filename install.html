Install

Linux

You can simply install using the following commands : 
git clone git://github.com/nbarbey/linear_operators.git
cd linear_operators
python setup.py install
Other operating systems are not supported.

Dependencies

List of dependencies:

  • numpy >= 1.3
  • scipy >= 0.8

Optional dependencies to define extra linear operators :

  • PyWavelets
  • fht (fast hadamard transform)
